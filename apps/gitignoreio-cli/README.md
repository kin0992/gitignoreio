# gitignoreio-cli

A command-line interface for the gitignore.io API.

## Installation

```bash
npm install -g gitignoreio-cli
```

## Usage

### List available templates

```bash
gitignoreio list
```

This command displays all available gitignore templates that you can use.

### Generate gitignore files (coming soon)

```bash
gitignoreio generate node python
```
